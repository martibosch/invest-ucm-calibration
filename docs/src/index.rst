Usage
-----

To install use pip:

.. code-block:: bash

    $ pip install invest-ucm-calibration

Or clone the repo:

.. code-block:: bash

    $ git clone https://github.com/martibosch/invest-ucm-calibration.git
    $ python setup.py install
